# 专为 Windows 移民打造
### 解决 Mac 窗口管理的 "水土不服"



<div align="center">

## 🌟 核心痛点与解决方案

**你是否也是刚从 Windows 转到 Mac 的用户？**
你是否习惯性地点一下任务栏图标想最小化窗口，结果 Mac 毫無反应？
你是否在寻找那个熟悉的 `Win + D` 回到桌面？

**GetBackMyWindows 就是你一直在寻找的答案。**
它极其纯粹，只做一件事：**把 Windows 的肌肉记忆带回 Mac。**

<br>

|     Windows 习惯     |           Mac 痛点           |      **GetBackMyWindows 方案**       |
| :------------------: | :--------------------------: | :----------------------------------: |
| 点击任务栏图标最小化 |      ❌ Dock 图标无反应       |   **✅ 点击 Dock 图标，立即最小化**   |
|  `Win + D` 显示桌面  | ❌ 需要复杂的 Mission Control | **✅ `⌃ + A` (自定义) 一键隐藏所有**  |
|   窗口管理直觉简单   |   ❌ 窗口层级混乱，难以理清   | **✅ 极速响应，逻辑直观，零学习成本** |

<br>
<br>

## 📦 极速安装

只需一步，即可拥有。

### 方式一：下载预编译版 (推荐)
<br>
<div align="center">
<a href="https://github.com/Avi7ii/GetBackMyWindows/releases">
    <img src="https://img.shields.io/badge/Download-Latest_Release-blue?style=for-the-badge&logo=github&logoColor=white&h=80" alt="Download Latest Release" height="40">
</a>
</div>
<br>
<br>

</div>

1. 下载最新的 `GetBackMyWindows.app.zip`
2. 解压并拖入 `Applications` 文件夹
3. 如果遇到权限提示，请在「系统设置 -> 隐私与安全性 -> 辅助功能」中勾选

<div align="center">

### 方式二：极客编译

</div>
```bash
git clone https://github.com/Avi7ii/GetBackMyWindows.git
cd GetBackMyWindows
chmod +x build.sh
./build.sh
```

## 🎮 使用指南

|          操作          |                     效果                     |
| :--------------------: | :------------------------------------------: |
| **左键点击 Dock 图标** | 最小化当前应用的所有窗口 (经典 Windows 体验) |
|   **快捷键 `⌃ + A`**   |       老板键 / 回到桌面 (隐藏所有窗口)       |
|     **菜单栏图标**     |       随时开启/暂停功能，配置开机自启        |
